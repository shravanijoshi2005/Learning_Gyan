String Interpolation
In JavaScript, string interpolation is a way to embed variables or expressions inside a string using template literals. Template literals are enclosed by backticks (`), and the variables or expressions are placed inside ${}.

Example:
const name = 'Peaches';
const age = 20;

console.log(`My name is ${name}, and I am ${age} years old.`);
Backticks (`) are used instead of quotes (' or ") to create the string.
${} is used to insert the value of a variable or expression inside the string.
